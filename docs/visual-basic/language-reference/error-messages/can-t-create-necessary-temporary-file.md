# N&#227;o &#233; poss&#237;vel criar arquivo tempor&#225;rio necess&#225;rio
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Ou a unidade que contém o diretório especificado pela variável de ambiente o TEMP está cheia , ou a variável de ambiente TEMP especifica uma unidade inválida ou somente leitura ou diretório.  
  
### Para corrigir este erro  
  
1.  Exclua arquivos da unidade, se cheia.  
  
2.  Especifique uma unidade diferente na variável de ambiente TEMP.  
  
3.  Especifique uma unidade válida para a variável de ambiente TEMP.  
  
4.  Remova a restrição somente leitura na unidade ou diretório especificados.
