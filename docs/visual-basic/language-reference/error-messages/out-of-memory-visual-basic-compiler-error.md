# Sem mem&#243;ria (erro do compilador do Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Foi requisitada mais memória do que a disponível.  
  
 **Identificação do erro:**  BC2004  
  
### Para corrigir este erro  
  
-   Feche aplicativos, documentos e arquivos fonte desnecessários.  
  
-   Elimine controles e formulários desnecessários pois menos são carregados de uma vez.  
  
-   Reduza o número de variáveis `Public`.  
  
-   Verifique espaço disponível em disco.  
  
-   Aumente a memória RAM disponível instalando memória adicional ou realocando memória.  
  
-   Certifique\-se de que foi liberada memória quando esta não for mais necessária.
