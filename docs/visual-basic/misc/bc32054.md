# &#39; &lt; nome do membro &gt; tem o mesmo nome que um par&#226;metro de tipo
Um membro de uma classe genérica ou estrutura tem o mesmo nome que um parâmetro de tipo de classe ou estrutura.  
  
 Cada variável, constante, enumeração, procedimento, propriedade, evento e delegado devem ter um nome distinto de cada parâmetro de tipo.  
  
 **ID do erro:** BC32054  
  
### Para corrigir este erro  
  
-   Altere o nome do membro da classe ou estrutura para que seja diferente de cada nome de parâmetro de tipo.
