# Nome XML esperado
Um literal XML ou propriedade de eixo XML foi especificada sem um nome.  
  
 **ID do erro:** BC31146  
  
### Para corrigir este erro  
  
-   Adicione um nome para o literal XML ou propriedade de eixo XML.
