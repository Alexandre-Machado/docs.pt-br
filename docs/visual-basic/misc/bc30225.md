# Instru&#231;&#227;o &#39;Option &lt; especificador &gt;&#39; s&#243; pode aparecer uma vez por arquivo
O `Option Compare`, `Option Explicit`, e `Option Strict` instruções podem ser usadas somente uma vez no mesmo arquivo de origem.  
  
 **ID do erro:** BC30225  
  
### Para corrigir este erro  
  
-   Remover a duplicação `Option` instrução.
