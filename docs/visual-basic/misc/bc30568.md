# O inicializador de matriz tem &lt; n&#250;mero &gt; muitos elementos
O inicializador de matriz contém muitos elementos.  
  
 **ID do erro:** BC30568  
  
### Para corrigir este erro  
  
-   Use `ReDim` para alterar o tamanho da matriz.
