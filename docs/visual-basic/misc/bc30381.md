# &#39;Finally&#39; s&#243; pode aparecer uma vez em uma instru&#231;&#227;o &#39;Try&#39;
`Finally` usado para concluir uma `Try...Catch...Finally` bloco; portanto ele pode aparecer apenas uma vez no final do bloco.  
  
 **ID do erro:** BC30381  
  
### Para corrigir este erro  
  
1.  Localize e remova a `Finally`.
