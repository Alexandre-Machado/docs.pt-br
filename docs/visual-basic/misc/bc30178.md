# O especificador est&#225; duplicado
Uma palavra\-chave aparece mais de uma vez na mesma declaração.  
  
 **ID do erro:** BC30178  
  
### Para corrigir este erro  
  
-   Remova o surgimento duplicado a palavra\-chave da declaração.
