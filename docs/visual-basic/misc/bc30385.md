# &#39;&lt; modificador &gt;&#39; n&#227;o &#233; v&#225;lido em uma declara&#231;&#227;o Delegate
Você tentou usar um modificador, como `Shared`, em uma declaração delegate. Delegados são objetos usados para chamar os métodos de outros objetos, definindo um construtor que é passado a especificação de um método do objeto. Não é válido para especificar um modificador em uma declaração delegate.  
  
 **ID do erro:** BC30385  
  
### Para corrigir este erro  
  
1.  Remova o modificador.
