# Este sistema n&#227;o tem suporte para a localidade chin&#234;s tradicional
Você tentou usar o `VbStrConv.TraditionalChinese` membro de enumeração, mas seu sistema não dá suporte para a localidade chinês tradicional.  
  
### Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade de chinês tradicional, ou execute o aplicativo em uma máquina que dá suporte a localidade.
