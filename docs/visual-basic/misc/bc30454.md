# Express&#227;o n&#227;o &#233; um m&#233;todo
O destino de sua chamada de método não é um método.  
  
 **ID do erro:** BC30454  
  
### Para corrigir este erro  
  
-   Certifique\-se de que digitado corretamente o nome do método.
