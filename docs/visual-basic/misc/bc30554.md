# &#39;&lt; nomenamespace &gt;&#39; &#233; amb&#237;guo
Você forneceu um nome que é ambíguo e, portanto, está em conflito com outro nome. O [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador não tem quaisquer regras de resolução de conflitos; você deve eliminar a ambigüidade por conta própria.  
  
 **ID do erro:** BC30554  
  
### Para corrigir este erro  
  
-   Qualificar totalmente o nome.
