# &#39;&lt; nomedoarquivo &gt;&#39; n&#227;o pode ser referenciado porque n&#227;o &#233; um assembly
Foi feita uma referência a um arquivo que não é um assembly .NET.  
  
 **ID do erro:** BC31519  
  
### Para corrigir este erro  
  
1.  Verifique se o nome e o local do assembly.  
  
2.  Verifique se o assembly não está danificado ou corrompido.
