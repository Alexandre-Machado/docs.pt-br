# Instru&#231;&#245;es &#39;Case&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Case` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30719  
  
### Para corrigir este erro  
  
-   Remover `Case` instruções de **imediato** janela.
