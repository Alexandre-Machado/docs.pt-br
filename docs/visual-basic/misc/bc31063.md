# M&#233;todo &#39;Set&#39; n&#227;o pode ter mais de um par&#226;metro
Mais de um parâmetro está definido para uma propriedade `Set` procedimentos.  
  
 **ID do erro:** BC31063  
  
### Para corrigir este erro  
  
-   Remover parâmetros extras de propriedade `Set` procedimentos.
