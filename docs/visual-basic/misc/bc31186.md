# &#39;:&#39; n&#227;o &#233; permitido
':' não é permitido. XML qualificado nomes não podem ser usados neste contexto.  
  
 Foi especificado um literal XML com um nome qualificado em um local inválido, por exemplo, um destino em um literal de instrução de processamento.  
  
 **ID do erro:** BC31186  
  
### Para corrigir este erro  
  
-   Remova o literal XML com o nome qualificado ou mova\-o para um local válido.  
  
     Para obter informações sobre quais caracteres são válidos em nomes XML, consulte [Extensible Markup Language \(XML\) 1.0](http://go.microsoft.com/fwlink/?LinkId=73927).
