# O n&#250;mero m&#225;ximo de erros foi excedido
Máximo de 100 erros de compilador foi atingido.  
  
 **ID do erro:** BC2020  
  
### Para corrigir este erro  
  
-   Diminua o número de erros gerados fazendo depuração e endereçamento isoladamente.
