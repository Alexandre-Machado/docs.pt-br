# Tipos n&#227;o est&#227;o dispon&#237;veis neste contexto
Você tentou fazer referência a um tipo em um `#If` diretiva.  
  
 **ID do erro:** BC30458  
  
### Para corrigir este erro  
  
1.  Remova a referência.
