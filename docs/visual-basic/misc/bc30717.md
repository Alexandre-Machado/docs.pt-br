# Instru&#231;&#245;es &#39;Try&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Try` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30717  
  
### Para corrigir este erro  
  
-   Remover `Try` instruções do código de depuração.
