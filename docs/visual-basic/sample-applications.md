# <a name="visual-basic-sample-applications"></a>Aplicativos de exemplo do Visual Basic
É possível usar o Visual Studio para baixar e instalar exemplos de aplicativos Visual Basic completos e empacotados na [Galeria de Códigos do MSDN](http://go.microsoft.com/fwlink/?LinkId=254185)  
  
 É possível baixar cada exemplo individualmente ou baixar um Pacote de Amostras que contém amostras relacionadas que compartilham uma tecnologia ou tópico. Você receberá uma notificação quando as alterações no código-fonte forem publicadas para qualquer exemplo baixado.
