# Controles DataRepeater (Visual Studio)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

O Visual Basic Power Packs <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controle é um contêiner rolável para repetido de controles que exibem dados, por exemplo, as linhas em uma tabela de banco de dados. Ele pode ser usado como uma alternativa para o <xref:System.Windows.Forms.DataGridView> controlar quando você precisar de mais controle sobre o layout dos dados.  
  
 Os controles PowerPack não estão mais incluídos no Visual Studio, mas você pode baixá\-los no [Centro de Download](http://www.microsoft.com/en-us/download/details.aspx?id=25169).
