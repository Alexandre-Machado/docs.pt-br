# CS1556 de erro do compilador
'Criar' especificado para método Main deve ser uma class ou struct válido  
  
 O [\/principal](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador foi passada um identificador que não era um nome de classe.
