# Compilador CS3000 de aviso (n&#237;vel 1)
Métodos com argumentos de variável não são compatíveis com CLS  
  
 Os argumentos usados no método de expõem recursos que não estão no Common Language especificações \(CLS\). Para obter mais informações sobre compatibilidade com CLS, consulte [Escrevendo código compatível com CLS](http://msdn.microsoft.com/pt-br/4c705105-69a2-4e5e-b24e-0633bc32c7f3).  
  
 O exemplo a seguir gera o aviso CS3000.  
  
```  
// CS3000.cs // compile with: /target:library // CS3000 expected [assembly:System.CLSCompliant(true)] public class Test { public void AddABunchOfInts( __arglist ) {}   // CS3000 }  
```
