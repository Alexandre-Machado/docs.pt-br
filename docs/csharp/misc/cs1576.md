# CS1576 de erro do compilador
O número de linha especificado para diretiva de \#line está ausente ou inválido  
  
 O compilador detectou um erro com o valor passado para o [\#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) diretiva.  
  
 O exemplo a seguir gera CS1576:  
  
```  
// CS1576.cs public class MyClass { static void Main() { #line "abc.sc"         // CS1576 // try the following line instead //#line  101 "abc.sc" intt i;  // error will be reported on line 101 } }  
```
