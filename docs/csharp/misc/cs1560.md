# CS1560 de erro do compilador
Nome de arquivo inválido especificado para diretiva de pré\-processamento. Nome do arquivo é muito longo ou não um nome de arquivo válido  
  
 O nome do arquivo que foi especificado com [\#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) excedeu MAX\_PATH \(256 caracteres\) ou a linha na qual `#line` foi encontrado excedidos 2000 caracteres.  
  
## Exemplo  
 O exemplo a seguir gera CS1560.  
  
```  
// cs1560.cs using System; class MyClass { public static void Main() { Console.WriteLine("Normal line #1."); #line 21 "MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890.txt"   // CS1560 } }  
```
