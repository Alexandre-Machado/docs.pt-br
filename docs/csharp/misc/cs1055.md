# CS1055 de erro do compilador
Um acessador add ou remove esperado  
  
 Se seu [evento](../../csharp/language-reference/keywords/event.md) não é declarado como um campo, ele deve definir ambos **Adicionar** e **Remover** funções do acessador.  
  
 O exemplo a seguir gera CS1055:  
  
```  
// CS1055.cs delegate void del(); class Test { public event del MyEvent { int i;   // CS1055 // uncomment accessors and delete previous line to resolve // add // { //    MyEvent += value; // } // remove // { //    MyEvent -= value; // } } public static void Main() { } }  
```
