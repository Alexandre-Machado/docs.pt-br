# &lt;para&gt; (Guia de Programa&#231;&#227;o em C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

## Sintaxe  
  
```  
<para>content</para>  
```  
  
#### Parâmetros  
 `content`  
 O texto do parágrafo.  
  
## Comentários  
 O \<para\> marca é para uso dentro de uma marca, como  [\<summary\>](../Topic/%3Csummary%3E%20\(C%23%20Programming%20Guide\).md),  [\<remarks\>](../../../csharp/programming-guide/xmldoc/remarks.md), ou  [\<returns\>](../../../csharp/programming-guide/xmldoc/returns.md)e permite que você adicione a estrutura do texto.  
  
 Compilar com  [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) a comentários de documentação do processo para um arquivo.

 Consulte  [\<summary\>](../Topic/%3Csummary%3E%20\(C%23%20Programming%20Guide\).md) para obter um exemplo do uso de \<para\>.
