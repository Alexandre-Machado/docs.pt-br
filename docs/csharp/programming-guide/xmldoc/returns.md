# &lt;returns&gt; (Guia de Programa&#231;&#227;o em C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

## Sintaxe  
  
```  
<returns>description</returns>  
```  
  
#### Parâmetros  
 `description`  
 Uma descrição de valor de retorno.  
  
## Comentários  
 O \<returns\> marca deve ser usada no comentário para uma declaração de método para descrever o valor de retorno.
