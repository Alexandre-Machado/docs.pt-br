# Compiler Error CS0616
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

as classes “não é uma classe de atributo  
  
 Foi feita uma tentativa de usar uma classe sem\-atributo em um bloco de atributo.  Todos os tipos de atributo precisam ser herdadas de <xref:System.Attribute?displayProperty=fullName>.  
  
## Exemplo  
 O exemplo a seguir produz CS0616.  
  
```  
// CS0616.cs  
// compile with: /target:library  
[CMyClass(i = 5)]   // CS0616  
public class CMyClass {}  
```  
  
## Exemplo  
 O exemplo a seguir mostra como você pode definir um atributo:  
  
```  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
