# Compiler Error CS0845
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Uma árvore de expressão lambda não pode conter um operador de união com um lado esquerdo literal nulo.  
  
 Como nulo em si não tem um tipo de união, o operador nulo não pode operá\-lo em.  
  
### Para corrigir este erro  
  
1.  Converter o literal nulo em um objeto.  
  
## Exemplo  
 O seguinte código gerenciado CS0845:  
  
```  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
