# Compiler Error CS0563
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Um dos parâmetros de um operador binário deve ser do tipo recipiente  
  
 A declaração do método para [sobrecarga do operador](../../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md) deve seguir algumas diretrizes.  
  
## Exemplo  
 O seguinte exemplo gera CS0563:  
  
```  
// CS0563.cs  
public class iii  
{  
    public static implicit operator int(iii x)  
    {  
        return 0;  
    }  
    public static implicit operator iii(int x)  
    {  
        return null;  
    }  
    public static int operator +(int aa, int bb)   // CS0563   
    // Use the following line instead:  
    // public static int operator +(int aa, iii bb)      
    {  
        return 0;  
    }  
    public static void Main()  
    {  
    }  
}  
```
